# 第二大脑 - 同步配置指南

## 本地 Obsidian 设置

### 1. 安装 Git 插件

1. 打开 Obsidian → 设置 → 社区插件
2. 关闭安全模式
3. 浏览 → 搜索 "Git"
4. 安装并启用

### 2. 配置 Git 插件

```json
{
  "commitMessage": "vault backup: {{date}}",
  "commitDateFormat": "YYYY-MM-DD HH:mm:ss",
  "autoSaveInterval": 0,
  "autoPullInterval": 0,
  "autoPullOnBoot": true,
  "disablePush": false,
  "pullBeforePush": true,
  "disablePopups": false,
  "disablePopupsForNoChanges": true
}
```

### 3. 初始化本地仓库

```bash
# 在 Obsidian 库目录下执行
cd D:\Obsidian\obsidian-vault
git init
git remote add origin https://github.com/Lizhiyu-foshan/obsidian-vault.git
git pull origin main
```

### 4. 日常使用

- **手动同步**：Ctrl+P → "Git: Pull" / "Git: Push"
- **自动同步**：可设置定时 pull/push
