## 项目概览

### 问题背景
定时任务"对话整理"执行时间从5秒增长到50秒，最终超时（60秒）。原因是扫描近3天的162个会话文件，随着文件增长必然再次超时。

### 解决方案
实现增量消息索引系统，维护消息级别的时间索引，每天只处理新增消息，避免全量扫描。
