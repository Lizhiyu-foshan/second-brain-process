## 使用说明

本日志由 `evolution_scanner.py` 自动生成，每日 23:00 更新。

### 统计项说明
- **错误数**: 当日记录在 ERRORS.md 中的错误数量
- **学习数**: 当日记录在 LEARNINGS.md 中的学习条目
- **改进数**: 当日 AGENTS.md 中记录的规则/流程改进
- **提交数**: 当日 Git 提交次数

### 趋势符号
- ↑N: 较昨日增加 N
- ↓N: 较昨日减少 N
- 持平: 与昨日相同
