```
└──────────────┬──────────────────────┘

┌─────────────────────────────────────┐

│  - 自动总结的旧对话摘要               │
│  - 保存在会话历史中                   │

└──────────────┬──────────────────────┘

│  - AGENTS.md → 行为规则              │
│  - SOUL.md → 人格设定                │
│  - 唯一跨会话存在的记忆层             │
└─────────────────────────────────────┘
```

## 核心设计原则

**"文件就是记忆"**

傅盛在文章里反复强调的那句话，也是 OpenClaw 的设计哲学：

> *"你认为你记住的东西，那是你的幻觉。不靠脑子，靠文件。"*

大语言模型的"我知道了" ≠ 人类的"我知道了"。模型只是在当前上下文里有这个信息，一旦刷新就没了。所以重要的事情必须写进文件。
